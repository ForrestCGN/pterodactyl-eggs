## Short note / Kurzhinweis

`VERSION_ID=latest` is currently not recommended for ATM11 because CurseForge does not reliably expose the server pack through the API. Use a specific ServerFiles File ID, for example `8357756` for ATM11 `0.2.0`.

`VERSION_ID=latest` ist aktuell für ATM11 nicht empfohlen, weil CurseForge das Serverpack nicht zuverlässig über die API meldet. Nutze eine konkrete ServerFiles File ID, zum Beispiel `8357756` für ATM11 `0.2.0`.
